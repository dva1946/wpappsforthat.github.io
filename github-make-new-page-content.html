Template Page Filename Definition
Design Objective Make it resonably simple:
  1. Easily add / edit pages.
  2. Always reloads the latest page and menu.
  3. Managing a menu system must be reasonable.
  4. Current website designs incorporates all-of-the-above.
  5. Hopefully what is outlined below and seeing the templates will make people interested in this solution for GitHub Pages.
Theory N/A A two new-page process:
  1. Duplicate github-newpage.html and rename to your-page-name.html
  2. Duplicate github-new-content.html and rename to github-your-page-name-content.html
  3. The github-header-template.html is used as-is for all web pages. Editing this page will be covered separately.
New Page Template github-newpage.html Primary page for generating a completely new page. It contains javascript and two content template pages. Only editing is editing the content page name.
Header Template Page github-header-template.html Contains the header image, navigation menu. Only editing required is add new menu items and change the header image.
Content Template Page github-new-content.html Contains the CONTENT of the page. This is where additions and changes are made to the page.
Style Sheet wpapps-style.css.html My latest style sheet file. It is / can continually change, but like all the template files it re-loads each time.
STEP ONE Create Two New Files Duplcate & rename two pages:
  1. github-newpage.html to [your-page-name].html
  2. github-new-content.html to github-[new]-content.html
  3. Example if page is MyPage: MyPage.html and github-MyPage-content.html
STEP TWO Edit new content page This is all content and you can do it anyway you wish. This does take some HTML & styling skills.
STEP THREE Add the page to a menu Edit github-header-template.html page and add it to a menu. If you are new to webpage creation, this could be a challenge. Sorry, I am not teaching these skills here.
STEP FOUR Copy all pages to GitHub Now copy the two new pages to your GitHub Website depository.
STEP FIVE Load website and run it Time for you to check out your new work.
NEED MORE HELP? Try W3schools.Com This is one of my go-to sites for HTML and CSS (styling) help. As I am a engineer and programmer HTML and styling are secondary skills for me, so I use what god gave us - Google.
Last Thought Have Fun If you are using GitHub Pages look at their website to learn more, if necessary.
