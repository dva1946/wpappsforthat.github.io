Installing and Using PayPal IPN for Wordpress

New - Oct 27th, 2022

IPN AppsForThat is functional, tested locally, via PayPal IPN Simulator and now production ready. The App was used live Oct 25th, 2022 at WoodWorkers Of Green Valley Recreation.

A new production only release is currently in the works:

  • It is only intended for PayPal Sandbox and Live use.
  • It will not include any local web page interface.
  • The goal is:
    • Download from Github
    • Configure (Sandbox or Live)
    • Upload to your website server
    • Test and and go live environment

Now What?

  • Initial Release:
    • Test model has Local and Sandbox Simulator testing.
    • Can also configure for Production use, but is a bit complex.
    • Still needs better documentation.
    • Does include MySQL interface.
    • Not dependent on any plugins.
    • Does require knowledge in Wordpress, PHP and MySQL.
  • Future Releases - More database interfaces:
    • WP Users.
    • Custom Member Payment App with its own MySQL table.
    • Basic reports.
    • Management tools (lots of work so no promises)
  • PayPal IPN App Objectives:
    • A solid base PayPal IPN App without adding plugins!
    • Expandable by the user community.
    • Contributions to the project from the user community.
  • A Closing Comment:
    • My primary objective is teach people how to fish!
